# 하노이 탑 (Tower of Hanoi)

## 📌 개요
**재귀 함수**를 이용한 하노이 탑 문제 구현입니다.  
원판 n개를 A 기둥에서 C 기둥으로 이동하는 전체 순서와 총 이동 횟수를 출력합니다.

---

## 🗂️ 파일 구조
```
📄 hanoi.c   - 하노이 탑 재귀 구현
```

---

## 🔧 알고리즘 원리

하노이 탑은 다음 **3단계 재귀**로 해결합니다.

```
hanoi(n, 시작, 보조, 목적지)
 ├── 1단계: n-1개를 시작 → 보조로 이동   (재귀)
 ├── 2단계: 가장 큰 원판을 시작 → 목적지로 이동
 └── 3단계: n-1개를 보조 → 목적지로 이동  (재귀)
```

### 시각적 구조 (n=3 기준)

```
초기 상태:         최종 상태:
A      B  C        A  B      C
|      |  |        |  |      |
=      |  |        |  |      =
==     |  |        |  |     ==
===    |  |        |  |    ===
```

---

## ⚙️ 함수 설명

| 함수 | 매개변수 | 설명 |
|------|---------|------|
| `hanoi(n, start, temp, arrive)` | n: 원판 수, start: 시작 기둥, temp: 보조 기둥, arrive: 목적지 기둥 | 재귀적으로 원판 이동 순서 출력 |

---

## 📋 실행 결과 (n=3)

```
원판 3개의 하노이 탑 이동 순서:
원판 1을 A에서 C로 이동 (이동 횟수: 1)
원판 2를 A에서 B로 이동 (이동 횟수: 2)
원판 1을 C에서 B로 이동 (이동 횟수: 3)
원판 3를 A에서 C로 이동 (이동 횟수: 4)
원판 1을 B에서 A로 이동 (이동 횟수: 5)
원판 2를 B에서 C로 이동 (이동 횟수: 6)
원판 1을 A에서 C로 이동 (이동 횟수: 7)
총 이동 횟수: 7
```

---

## 📊 시간 복잡도

| 원판 수 (n) | 이동 횟수 |
|------------|---------|
| 1 | 1 |
| 2 | 3 |
| 3 | 7 |
| 4 | 15 |
| n | **2ⁿ - 1** |

> 이동 횟수는 항상 **2ⁿ - 1** 번으로, 시간 복잡도는 **O(2ⁿ)** 입니다.
