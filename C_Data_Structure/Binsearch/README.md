# 이진 탐색 트리 (Binary Search Tree)

## 📌 개요
**재귀 함수**를 이용한 이진 탐색 트리(BST) 구현입니다.  
삽입, 탐색, 삭제 연산을 구현하고 10개의 노드를 이용해 동작을 검증합니다.

---

## 🗂️ 파일 구조
```
📄 Binsearch.c   - 이진 탐색 트리 구현 및 테스트 메인
```

---

## 🔧 노드 구조

```c
typedef struct Node {
    int data;           // 데이터
    struct Node* left;  // 왼쪽 자식 (작은 값)
    struct Node* right; // 오른쪽 자식 (큰 값)
} Node;
```

### BST 성질
```
        [50]
       /    \
    [30]    [70]
    /  \    /  \
  [20][40][60][80]
  /      \    \
[10]    [35]  [65]

- 왼쪽 서브트리의 모든 값 < 루트
- 오른쪽 서브트리의 모든 값 > 루트
- 중위 순회 시 오름차순 출력 보장
```

---

## ⚙️ 함수 설명

| 함수 | 설명 | 시간복잡도 |
|------|------|-----------|
| `alloc_node(data)` | 새 노드 생성 및 초기화 | O(1) |
| `search(root, target)` | target 탐색, 성공 시 노드 포인터 반환 | O(log n) ~ O(n) |
| `insert(root, data)` | data 삽입 (중복 불가) | O(log n) ~ O(n) |
| `find_min(root)` | 서브트리에서 최솟값 노드 반환 | O(log n) ~ O(n) |
| `delete(root, target)` | target 삭제 (3가지 케이스 처리) | O(log n) ~ O(n) |
| `destroy_tree(root)` | 트리 전체 메모리 해제 | O(n) |
| `inorder(root)` | 중위 순회로 오름차순 출력 | O(n) |
| `print_tree(root, depth)` | 트리 구조 시각화 출력 | O(n) |
| `search_and_print(root, target)` | 탐색 결과 메시지 출력 | O(log n) ~ O(n) |

## 🔍 삭제 연산 3가지 케이스

```
case 1: 자식 없음 (단말 노드) → 그냥 삭제
---------------------------------------------
  삭제 전)  [20]        삭제 후)  [20]
            /
          [10]  ← 삭제


case 2: 자식 1개 → 자식 노드로 대체
---------------------------------------------
  삭제 전)  [30]        삭제 후)  [30]
            /                     /
          [20]  ← 삭제          [35]
              \
              [35]


case 3: 자식 2개 → 중위 후계자(오른쪽 서브트리 최솟값)로 대체
---------------------------------------------
  삭제 전)    [30]  ← 삭제     삭제 후)    [35]
             /    \                        /    \
           [20]  [40]                   [20]  [40]
                 /
               [35] ← 중위 후계자
```

## 📊 시간 복잡도

| 연산 | 평균 | 최악 (편향 트리) |
|------|------|----------------|
| 탐색 | O(log n) | O(n) |
| 삽입 | O(log n) | O(n) |
| 삭제 | O(log n) | O(n) |

> 최악의 경우는 오름차순/내림차순으로 삽입할 때 트리가 한쪽으로만 편향되는 경우입니다.  
> 이를 해결하려면 **AVL 트리** 또는 **Red-Black 트리**를 사용합니다.
